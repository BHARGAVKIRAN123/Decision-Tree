# Decision Tree Classification Project

This project demonstrates the implementation of the Decision Tree algorithm in Machine Learning using Python and Scikit-learn.

## Concepts Covered
- Decision Tree Classifier
- Data Preprocessing
- Train-Test Split
- Model Training
- Prediction
- Accuracy Evaluation

## Tools Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## Project Description
Decision Tree is a supervised machine learning algorithm used for classification and regression tasks.  
In this project, I trained a Decision Tree model on a dataset, tested it on unseen data, and evaluated its performance.

This exercise helped me understand how a model makes decisions by splitting data into branches based on conditions.

## Learning Outcome
- Understood the working of Decision Tree algorithm
- Practiced training and testing a model
- Learned how to evaluate classification performance
- Completed an assessment based on the topic
